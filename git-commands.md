"## Основные команды git\n\n* init: Инициализация нового репозитория.\nПример: git init\n\n* clone: Клонирование существующего репозитория.\nПример: git clone <url>\n\n* add: Добавление изменений в индекс.\nПример: git add <file>\n\n* commit: Создание коммита с сообщением.\nПример: git commit -m \"сообщение\"\n\n* push: Отправка изменений в удаленный репозиторий.\nПример: git push origin <branch>\n\n* pull: Получение изменений из удаленного репозитория и объединение с локальной веткой.\nПример: git pull origin <branch>"
